## FETCH SIM - GAZEBO
This repo contains simulation of fetch in Gazebo with moveit-2 setup. 
* This is intended on to work with ROS 2.

### 1. Setup
```
mkdir -p fetch_ws/src
cd fetch_ws/src
git clone git@github.com:Team-I-ORL/fetch_gazebo.git
cd ..
colcon build
```
